## 使用方法

目前完整功能可下载安装演示app，效果一样
导入HBuilerx 3+ 后，直接真机调试运行即可看到效果，后端演示接口目前可用
你可参照当前已有的接口，使用开源小说cms系统仿写



## api接口

>	getHot 获取人气热门图书
>	
>	getNewest 获取最新图书
>	
>	getBanners 获取首页banner
>	
>	getList 获取全部分类
>	
>	getBookList 根据分类获取图书列表
>	
>	getRankList 获取排行榜数据
>	
>	getDetail 获取图书详情信息
>	
>	getMuluList 获取图书目录列表
>	
>	getText 获取某个章节的正文信息
>	
>	getSearch  搜索图书
